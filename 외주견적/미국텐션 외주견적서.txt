프로젝트명: TENtion – 소셜 슬롯 매칭 앱 백엔드 개발

[프로젝트 개요]
React Native(Expo)로 제작된 프론트엔드(App.js 완성본)가 있으며,
백엔드 + 로그인 + DB 연동 + 기본 운영툴 구현이 필요한 상태입니다.

[개발 범위]
1) 인증/로그인
- 이메일 로그인 (필수)
- Google OAuth 가능하면 추가
- Access/Refresh Token 방식
- 프로필(닉네임/바이오/아바타 URL) 저장/조회

2) 슬롯(Slot) 기능 백엔드
- 슬롯 생성
- 슬롯 목록 조회 (필터: 도시, 카테고리, 시간대, 거리, 정렬)
- 슬롯 참여 / 취소 (join / leave)
- 내가 만든/참여한 슬롯 조회

3) 데이터베이스
- PostgreSQL / Supabase / Firebase 중 하나 추천
- 필요한 테이블: users / profiles / slots / slot_attendees

4) 운영툴(간단)
- 웹 기반 관리자 페이지
- 유저 목록
- 슬롯 목록
- 신고 관리 기본 구조

5) 서버 배포
- AWS / GCP / Supabase 중 선택
- HTTPS 적용
- API 문서 제공(Swagger 또는 Markdown)

[프론트 연동 포인트]
프론트는 제가 제공.
다음 API 엔드포인트 연동 필요:
- POST /auth/login
- POST /auth/register
- GET /me/profile
- PATCH /me/profile
- GET /slots
- POST /slots
- POST /slots/:id/join
- POST /slots/:id/leave
- GET /me/slots

[희망 일정]
- 개발 기간: 2~3주 이내 (10~15일 작업량 기준)
- 가능 일정 제안해주세요.

[예산 범위]
- 300~600만 원 사이 희망
- 개발자님의 금액/일정 제시해주세요.

[필수 포함 조건 — 계약 조항]
1) 비밀유지(NDA)
- 프로젝트 관련 모든 정보는 외부 공유 및 재사용 금지
- 작업 종료 후에도 비밀유지 유지

2) 저작권 및 소유권
- 산출되는 모든 코드/설계/문서/DB 구조는 의뢰인(저)에게 100% 귀속
- 개발자는 포트폴리오·타 프로젝트 재사용 금지

3) 리스크 및 A/S
- 작업 범위 내 버그는 최소 30일 무상 수정
- 외부 라이브러리 사용 시 사전 고지 필수

[추가 요청]

- DB ERD 제공
